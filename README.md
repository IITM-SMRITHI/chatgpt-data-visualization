Employee Performance Analysis
Business Case
Retail company workforce analysis for strategic planning and resource allocation decisions.

Files in this Repository
employee_data.csv - Dataset containing 100 employee records across 6 departments
employee_analysis.py - Python script for data analysis and visualization
employee_analysis.html - Complete analysis report with embedded visualization
department_distribution.png - Department distribution histogram
Key Findings
R&D Department Frequency Count: 22 employees (22% of workforce)
Total Employees: 100
Number of Departments: 6 (Finance, HR, IT, Marketing, R&D, Sales)
R&D shows high performance metrics and employee satisfaction

How to Run
Install dependencies:

pip install pandas matplotlib seaborn
Run the analysis:

python employee_analysis.py
Open employee_analysis.html in a browser to view the complete report
